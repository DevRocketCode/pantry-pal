# pantry-pal
A web app that lists yummy recipes. Written in TypeScript, Express, and Node.js.

## Getting Started

1. Clone this repo
2. Run `npm install`
3. Run `npm start` to start the server
4. Open your browser to [http://localhost:3000](http://localhost:3000)
